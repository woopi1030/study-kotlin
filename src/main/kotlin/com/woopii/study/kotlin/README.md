## 장점

1. 가독성 
2. 지원 도구(JetBrains)
3. 다중 패러다임
   - 명령형 프로그래밍
   - 함수형 프로그래밍
   - 객체 지향 프로그래밍
4. 다중 플렛폼
   - JVM : 소스코드를 JVM바이트코드로 컴파일 하고, JVM에서 실행할 수 있다.
   - 안드로이드 : ART라는 자체 런타임이 있다.
   - 자바 스크립트 : 웹 브라우저에서 실행 가능
   - 네이티브 바이너리 : 특정 플렛폼과 CPU에 맞는 기계어 코드를 생성할 수 있다.

## 특징

1. 자바와의 상호 운용성
   - JVM에서의 완벽한 동작 유무 
   - 자동 변환 기능
2. 빈 값 표현 방식
   - 널 오류를 발생시킬 수 있는 연산을 컴파일 시점에 금지한다.
